# Getting Started
To start Metro, run the following command from the _root_ of your React Native project:
```bash
# using npm
npm start
# OR using Yarn
yarn start
```
## Step 2: Start your Application
### For Android
```bash
# using npm
npm run android
# OR using Yarn
yarn android
```
